# GameOfLife_Flask
В данном реппозитории хранится проект "Вселенная Конвея"

Инструкция для Windows пользователей (другие сами разберутся). Если у вас синее окошко и маршрут начинается с PS - это PowerShell, там другие команды нужны

Скачиваем проект в отдельную папку. Заходим в неё

В адресной строке папки вводим cmd - открывается командная строка сразу по нужному маршруту. Должно открыться черное окно - если открылось синее - это PowerShell, закройте его. Используйте горячие клавиши Win+R откроется маленькое окошко с заголовком Выполнить, вбейте там cmd и нажмите Enter - откроется черное окно. Скопируйте абсолютный путь до проекта из адресной строки в папке и введите команду cd /d (здесь пробел) ваш путь до проекта (вроде D:\User\Dekstop\GameOfLife)

Создаем виртуальное окружение командой python -m venv .venv

Активируем виртуальную среду командой .venv\Scripts\activate

Устанавливаем библиотеки командой pip install -r requirements.txt

Заходим в папку game_of_life командой cd game_of_life

Запускаем проект командой flask run

Открываем в браузере http://127.0.0.1:5000

Постарайтесь найти секретное "правило соседей". Удачи
